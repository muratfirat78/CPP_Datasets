
The AI4I 2020 Predictive Maintenance Dataset (ai4i2020.csv) is a synthetic dataset that reflects real predictive maintenance data encountered in industry.
The purpose of this data set is to predict machine failure.
See the data description below.

The data set is collected from UCI, more information can be found on the web site.
Follow the provided forms for the solution of Task 1.
